# 衡录「Signal」视觉方案

> **状态：现行方向（2026-07）**  
> 已废弃「墨衡」水墨 / 印章 / 宣纸文人风（见文末）。

## 概念

**Signal（信号）**：AI 时代的个人站点气质——炫酷、简约、现代化。  
品牌「衡录」用几何无衬线大字做英雄级信号；界面靠清晰层级与克制动效，而不是装饰性纸纹。

## 关键词

| 要                                    | 不要                     |
| ------------------------------------- | ------------------------ |
| 近黑 / 冷灰画布 + 青绿 / 天蓝点缀     | 紫粉渐变、奶油衬线套餐   |
| Outfit + IBM Plex Sans + Noto Sans SC | Inter 默认堆、宋体印章感 |
| 微网格 + 柔光斑                       | 墨晕、宣纸噪点、竖刻度   |
| 品牌大字主导首屏                      | 弱品牌 + 堆统计/卡片墙   |
| `prefers-reduced-motion` 降级         | 无意义循环炫技           |

## 色板（CSS 令牌）

浅色：`#f3f5f8` 底、`#0b1220` 字、`#0d9488` 主强调、`#0284c7` 辅强调。  
暗色：`#070a0f` 底、`#e8eef7` 字、`#2dd4bf` / `#38bdf8` 强调。

令牌经 `global.css` 的 `@theme` 映射为 `text-hl-fg`、`bg-hl-accent` 等 Tailwind 工具类；`html[data-theme]` 切换浅暗。

## 动效（2–3 处标杆）

1. 英雄区文案错落入场 `rise`
2. 背景光斑缓动 `glow-pulse`
3. 导航/按钮 hover 微位移或下划线（无 reduce-motion 时）

## 无障碍（ux-audit）

- 跳过链接、可见焦点环、语义导航
- 对比度浅/暗均按 AA 取向
- 主题/语言切换有 `aria-live` 反馈
