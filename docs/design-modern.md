# vv Studio 视觉方案

> **状态：现行方向（2026-07）**  
> 已废弃「墨衡」水墨风与早期青绿 Signal 模板感。  
> 品牌统一为 **vv**。

## 概念

**vv Studio**：可交付的个人站产品气质——炫酷、简约、现代。  
巨型字标 **vv** 是首屏唯一英雄信号；设置与偏好迁到「关于」，首屏只做品牌与阅读入口。

## 关键词

| 要                                  | 不要                     |
| ----------------------------------- | ------------------------ |
| 石墨画布 + 信号橙强调               | 紫粉渐变、奶油衬线套餐   |

## 色板（CSS 令牌）

浅色：`#f2f3f5` 底、`#12141a` 字、`#e2411a` 主强调、`#2563eb` 辅强调。  
暗色：`#0a0b0e` 底、`#f4f5f7` 字、`#ff6a3d` / `#60a5fa` 强调。

令牌经 `global.css` 的 `@theme` 映射为 `text-hl-fg`、`bg-hl-accent` 等；`html[data-theme]` 切换浅暗。

## 动效（2–3 处标杆）

1. 英雄区文案错落入场 `rise`
2. 品牌下划线 `brand-underline`
3. 背景光斑缓动 `glow-pulse`

## 信息架构（产品感）

- 顶栏：主链 4 项 +「更多」折叠；搜索常驻
- 首页：品牌首屏 → 最近写作 → 频道入口
- **偏好**：右下角悬浮「设置」抽屉（主题 / 语言 / 阅读辅助），不占独立页
- **回顶**：滚动超过一屏后出现 ↑ 按钮
- 关于：站介绍 + 今日栈（非设置页）
- 文章：大标题 + 编辑式 meta + 窄栏正文 + 粘性目录

## 无障碍（ux-audit）

- 跳过链接、可见焦点环、语义导航
- 对比度浅/暗均按 AA 取向
- 主题/语言切换有 `aria-live` 反馈
